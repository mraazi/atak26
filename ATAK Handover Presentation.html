Apa itu ATAK Webapp?

Satu fail HTML tunggal yang mengandungi keseluruhan sistem pengurusan akademi. Tidak perlu pemasangan — buka sahaja dalam pelayar web.

👤 Pengurusan Pelajar Tambah, sunting, dan pantau rekod pelajar serta kehadiran.
💳 Pembayaran Billplz Terbitkan invois dan terima bayaran dalam talian melalui FPX, kad, e-dompet.
📚 Portal Pelajar Pelajar log masuk untuk lihat kelas, tugasan, dan bayar yuran.
📊 Laporan & Kemajuan Rekod bacaan harian, kehadiran, dan laporan perkembangan tajwid.
📁 Sandaran Data Eksport semua data ke Google Drive. Import semula pada mana-mana peranti.
🔒 Kawalan Akaun Admin cipta akaun untuk setiap pelajar dan guru. Tiada pendaftaran sendiri.
🔑

Cara Akses & Log Masuk

Buka fail ATAK Webapp _Standalone_.html dalam pelayar web (Chrome/Safari/Edge).

⚠️
Data disimpan dalam localStorage pelayar semasa. Jika tukar pelayar atau peranti, import fail sandaran terlebih dahulu.

Slide 5 / 10

Proses: Daftar Pelajar Baru

Admin → tab Pelajar → klik 👤 Daftar Pelajar Baru

Wizard 3 langkah — semua selesai dalam satu modal.

1
Maklumat Pelajar Nama penuh, kelas, umur, nama penjaga, no. telefon penjaga
2
Akaun Login ID pengguna (username) dan kata laluan — untuk pelajar log masuk
3
Invois Pertama Keterangan, jumlah (RM), tarikh akhir — atau langkau untuk buat kemudian
✅ Klik Daftarkan — rekod pelajar, akaun login dan invois pertama dibuat sekaligus.

Apa yang dibuat secara automatik

REKOD PELAJAR

Nama, kelas, penjaga disimpan dalam senarai pelajar. Avatar auto-jana dari inisial nama.

AKAUN LOGIN

Username & kata laluan disimpan. Pelajar boleh log masuk serta-merta.

INVOIS (jika tidak dilangkau)

Invois terus muncul dalam tab Pembayaran pelajar berkenaan.

⚠️
Berikan username & kata laluan kepada pelajar / ibu bapa selepas daftar.

Sandaran Data — Google Drive

Semua data disimpan dalam localStorage pelayar. Eksport secara berkala dan simpan ke Google Drive sebagai sandaran.

💾 Cara Eksport

1
Admin → ⚙ Tetapan → Sandaran DataTatal ke bahagian bawah tetapan
2
Klik 💾 Eksport DataFail atak-data-YYYY-MM-DD.json akan dimuat turun
3
Simpan ke Google DriveSeret fail ke folder Google Drive anda

📂 Cara Import (Pulih Data)

1
Muat turun fail JSON dari Google DrivePilih fail sandaran terkini
2
Admin → ⚙ Tetapan → 📂 Import DataPilih fail JSON yang dimuat turun
3
Data dipulihkan serta-mertaTiada muat semula halaman diperlukan
